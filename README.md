# 📰 NEWS-GUIDE - NEXT.JS - APP

🌟 Overview
A modern News Web Application built using Next.js that delivers real-time news with a fast and responsive UI.
It supports search, multiple languages, and smooth infinite scrolling for a seamless user experience.


## 📌 Features
* 📰 Real-time API Data Fetching 
* 🗞️ Latest news from multiple categories
* 🔍 Search functionality for custom queries
* 🌐 Multi-language support
* ⚡ Fast and optimized with Next.js (SSR/CSR)
* ♾️ Limited scrolling for better UX
* 📱 Fully responsive design
* 🖼️ Fallback image support for missing media 



## 🛠️ Tech Stack

* **Frontend:** Next.js, React.js
* **Styling:** CSS / Bootstrap  (update if needed)
* **API:** News API
* **Routing:** Next.js App Router
* **State Management:** React Hooks
* **Javascript** 
* **Backend:** API Integration, REST API
